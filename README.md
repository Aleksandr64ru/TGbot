# 🚀 Crypto Telegram Bot

Простой и удобный Telegram-бот для отслеживания криптовалют в реальном времени.

---

## 📊 Возможности

* 💰 **Цена криптовалют**

  * `/price BTC` — узнать цену любой монеты

  * показывает изменение за 24ч и капитализацию

  * встроенный ASCII-график 📈

* 🔝 **Топ криптовалют**

  * `/top` — топ-10 монет по капитализации

* 📰 **Новости**

  * `/news` — последние новости крипторынка

---

## ⚙️ Технологии

* Python 3.10\+

* aiogram 3.x

* aiohttp

* CoinGecko API

## 🚀 Запуск проекта

### 1. Клонировать репозиторий

```
git clone <your-repo-url>
cd TGbot
```

### 2. Установить зависимости

```
pip install -r requirements.txt
```

### 3. Установить токен бота

Создай переменную окружения:

```
BOT_TOKEN=your_telegram_bot_token
```

---

### 4. Запуск

```
python src/main.py
```
